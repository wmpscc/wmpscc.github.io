Windows下安装Tensorflow-Gpu

本文总共809个字 | 您是第位看到它们的小伙伴

文章时间：2017/12/1
Tensorflow：https://www.tensorflow.org/install/install_windows
CUDA Toolkit 8.0：https://developer.nvidia.com/cuda-80-ga2-download-archive
cuDNN：https://developer.nvidia.com/rdp/cudnn-download
Python version:3.6.2 此版本为tensorflow目前支持的最新版本，3.6.3暂时不支持
本文安装的是Tensorflow-gpu版本

安装：

（一）：安装好python3.6.2和Visual studio2017，2015以上版本即可。不安装会出错

（二）：安装CUDA 8.0（9.0暂时不支持，上面有链接）

（三）：安装cuDNN（上面有链接，下载解压将文件覆盖到~cuda/dev目录）

（四）：配置系统环境变量


需要添加CUDA安装目录和bin文件夹目录
例如：
C:\dev\cuda
C:\dev\cuda\bin

（五）：pip3安装Tensorflow

最好使用Windows PowerShell管理员运行该命令：
pip3 install –upgrade tensorflow-gpu
（自行安装pip3，注意需要将更新源设置为国内的，或者开启科学上网）

（六）：测试

输入

如果输出：Hello, TensorFlow!
表明安装成功。
祝安装顺利~！
